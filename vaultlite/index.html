Veeva Vault–Inspired Platform on AWS (Free Options)

VaultLite is a serverless document platform MVP demonstrating secure storage, metadata, versioning, role-based access, approval workflows, and audit trails — using AWS services that are Free Tier / low-cost friendly.

View Architecture Step-by-Step Build GitHub

What it Includes

  • Secure document upload/download (S3 + pre-signed URLs)
  • Metadata + listing (DynamoDB)
  • RBAC (Cognito groups: Admin/Approver/Contributor/Viewer)
  • Workflow: Draft → Review → Approved/Rejected (Step Functions)
  • Audit trail per document (DynamoDB AuditEvents + CloudTrail Event History)

Tech Stack

S3DynamoDBLambda API GatewayCognitoStep Functions CloudWatchCloudTrail

Goal: demonstrate real platform patterns — auth, least privilege IAM, workflow orchestration, auditability, and API-first design.

Executive Summary

This project shows how to design a “Vault-like” platform using serverless building blocks: S3 for document storage, DynamoDB for metadata and audit events, Cognito for identity and groups, API Gateway + Lambda for the API layer, and Step Functions for lifecycle workflows.

Architecture

High-Level Flow

User → Cognito (JWT) User → API Gateway → Lambda Lambda → DynamoDB (Documents, AuditEvents) Lambda → S3 (pre-signed upload/download URLs) Workflow actions → Step Functions → status updates + audit events Logs → CloudWatch AWS account activity → CloudTrail Event History

Beginner Build Steps

0) Cost Safety

Set AWS Budgets alerts ($1/$5/$10). Avoid NAT Gateways and heavy exports early.

1) Setup Tools

AWS CLI + Node.js LTS + CDK (or Terraform). Configure AWS credentials.

2) Data Model

S3 for files. DynamoDB for Documents metadata and AuditEvents (append-only).

3) Cognito Auth

User Pool + App Client + groups for RBAC. API Gateway JWT authorizer.

4) API Layer

API Gateway routes → Lambda handlers. Use pre-signed URLs for upload/download.

5) Workflow

Step Functions orchestrates Draft→Review→Approved/Rejected and writes audit events.

Lambda: Create Document (Node.js)

import { DynamoDBClient } from "@aws-sdk/client-dynamodb"; import { DynamoDBDocumentClient, PutCommand } from "@aws-sdk/lib-dynamodb"; import { randomUUID } from "crypto"; const ddb = DynamoDBDocumentClient.from(new DynamoDBClient({})); export const handler = async (event) => { const requestId = event.requestContext?.requestId || randomUUID(); const actor = event.requestContext?.authorizer?.jwt?.claims?.sub || "anonymous"; const body = JSON.parse(event.body || "{}"); const docId = randomUUID(); const item = { pk: `DOC#${docId}`, sk: "META", title: body.title || "Untitled", docType: body.docType || "General", status: "Draft", owner: actor, createdAt: new Date().toISOString(), updatedAt: new Date().toISOString(), latestVersionId: null, tags: body.tags || [] }; await ddb.send(new PutCommand({ TableName: process.env.DOCS_TABLE, Item: item, ConditionExpression: "attribute_not_exists(pk)" })); console.log(JSON.stringify({ level: "INFO", requestId, action: "DOC_CREATE", docId: item.pk, status: item.status })); return { statusCode: 201, headers: { "content-type": "application/json" }, body: JSON.stringify({ docId, status: item.status }) }; };

Audit Logging Pattern (Append-only)

const auditEvent = { pk: `DOC#${docId}`, sk: `TS#${new Date().toISOString()}#EVT#${crypto.randomUUID()}`, actor, action: "DOC_APPROVE", fromStatus: "Review", toStatus: "Approved", ip: event.requestContext?.identity?.sourceIp, userAgent: event.headers?.["user-agent"] }; // Put into AuditEvents table (no updates, no deletes) await ddb.send(new PutCommand({ TableName: process.env.AUDIT_TABLE, Item: auditEvent }));

API Endpoints

POST /docs → create document metadata (Draft) GET /docs → list docs (status/owner filters) GET /docs/{docId} → doc details + latest version POST /docs/{docId}/upload-url → pre-signed S3 upload URL POST /docs/{docId}/submit → Draft → Review (starts workflow) POST /docs/{docId}/approve → Review → Approved (Approver only) POST /docs/{docId}/reject → Review → Rejected (Approver only) GET /docs/{docId}/download-url → pre-signed S3 download URL
