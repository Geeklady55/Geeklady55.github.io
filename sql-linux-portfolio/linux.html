Recommended Artifacts to Add

commands.md

Document the exact Linux commands used, why they matter, and what signals you looked for.

scripts/

Store Bash or Python utilities with comments, safe defaults, and a structure that is easy to review.

errors + fixes

Capture the problem, the evidence, the fix, and how you validated that the issue was really resolved.

Tip: Put screenshots and supporting files inside each project folder and use relative links like ./file-name for reliability.
