Recommended Artifacts to Add

queries.sql

Full query pack with comments. Keep the page sample short, link the full file for download.

schema.sql

Minimal schema snippets (tables/keys/indexes) so reviewers understand the query context.

explain-plan.md

Before/after EXPLAIN output and what changed. Include measurable timing/rows when possible.

Tip: Keep filenames lowercase and avoid spaces to prevent GitHub Pages path issues.
